MODE CHOICE MODEL SENSITIVITIES

This page highlights the factors that influence accessibility scores and mode choice model outputs. The roster of key variables includes general indications of their impact on model results. For more details on mode choice model variables, see the mode choice model report

Four tables are presented in the tabs below:

  • Access scores are sensitive to the locations of jobs and households as well as the travel networks for each analyzed mode. This table focuses on the network attributes that influence accessibility scores. Since these scores are used in the mode choice models, these factors have a downstream impact on estimated mode shares.
  • Production end mode choice models combine access scores with household characteristics to estimate mode choice for the HBW, HBO, and HBSch purposes.
  • Attraction end mode choice models focus primarily on accessibility scores, with some consideration of the characteristics of households that can access a given zone alongside built environment variables.
  • Non-home-based mode choice models rely on access scores and the outputs of the attraction end mode choice models. When analyzing NHB trips, the mode of travel used to reach the non-home location is an important determinant of the likely mode of travel.
